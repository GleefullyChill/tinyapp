## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- morgan
- mocha
- chai
- nodemon

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `run npm` command.
- Set all required PORT settings to '4321', or change the PORT # in server_express.js
